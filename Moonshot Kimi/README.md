# Moonshot Kimi

System prompts, behavior patterns, and integration guides for Moonshot AI's Kimi models (K2, K2.5, K2-thinking).

## Files

- **system-behavior.md** - Model behavior patterns and characteristics
- **tool-use-patterns.md** - Tool calling examples and best practices
- **api-integration.md** - API usage and OpenAI SDK compatibility
